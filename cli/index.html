CLI Reference

This page provide a reference documentation to our cli

babylon

CLI used for cloud interactions between CosmoTech and multiple cloud environment

Usage:

babylon [OPTIONS] COMMAND [ARGS]...

Options:

Name Type Description Default
--verbosity, -v text Either CRITICAL, ERROR, WARNING, INFO or DEBUG INFO
-e, --environment text N/A .
-t, --template text N/A None
--help boolean Show this message and exit. False

Subcommands

  • api: Group handling communication with the cosmotech API
  • environment: Command group handling local environment information

api

Group handling communication with the cosmotech API

Usage:

babylon api [OPTIONS] COMMAND [ARGS]...

Options:

Name Type Description Default
--help boolean Show this message and exit. False

Subcommands

  • connector: Create a new connector in the API using FILENAME as a template
